# 🐺 LoboBus - Sistema de Navegación y Seguimiento de Transporte Universitario

## 📱 Descripción
LoboBus es una aplicación iOS desarrollada para la Benemérita Universidad Autónoma de Puebla (BUAP) que proporciona un sistema inteligente de navegación y seguimiento del transporte universitario. La aplicación permite a los estudiantes y personal universitario conocer en tiempo real la ubicación de los autobuses, calcular rutas óptimas y recibir notificaciones sobre la llegada del transporte.

## 🚀 Características Principales

### 🗺️ Navegación y Rutas
- Cálculo de rutas más cortas usando el algoritmo de Dijkstra
- Visualización de rutas en mapas interactivos
- Optimización de rutas considerando múltiples variables
- Sistema de predicción de tiempos de llegada

### 📍 Seguimiento en Tiempo Real
- Monitoreo en vivo de la ubicación del autobús
- Visualización de paradas en mapa interactivo
- Cálculo de tiempos estimados de llegada
- Sistema de notificaciones para alertar cuando el autobús está cerca

### 👤 Sistema de Usuario
- Autenticación segura de usuarios
- Perfil personalizado con información del estudiante
- Historial de rutas frecuentes
- Gestión de sesiones de usuario

## 🛠️ Tecnologías Utilizadas

### Frontend
- SwiftUI
- MapKit
- CoreLocation
- Swift

### Algoritmos y Estructuras de Datos
- Algoritmo de Dijkstra para rutas más cortas
- Grafos dirigidos para representación de rutas
- Colas de prioridad para optimización
- Tablas hash para almacenamiento eficiente

## 📋 Requisitos del Sistema

### Para Desarrollo
- Xcode 13.0 o superior
- iOS 15.0 o superior
- Swift 5.5 o superior
- CocoaPods (para gestión de dependencias)

### Para Usuario Final
- iOS 15.0 o superior
- Conexión a Internet
- Servicios de ubicación activados

## 🚀 Instalación

1. Clona el repositorio:
```bash
git clone https://github.com/tu-usuario/LoboBus.git
```

2. Navega al directorio del proyecto:
```bash
cd LoboBus
```

3. Instala las dependencias:
```bash
pod install
```

4. Abre el archivo `LoboBus.xcworkspace` en Xcode

5. Compila y ejecuta el proyecto

## 📱 Uso

1. Inicia sesión con tus credenciales universitarias
2. Selecciona tu parada actual
3. Elige tu destino
4. La aplicación calculará la ruta más corta
5. Recibe notificaciones sobre la llegada del autobús
